# PROTAL WEBSITE SERVICE

### 1. Pre-Requisites

- Install NodeJS 16+
- Install Nodemon

### 2. Tech Stack

- Express
- PostgreSQL
- Sequelize
- Express Validator

### 3. Getting Started

#### 3.1 Clone Project

- git clone url -b master

#### 3.2

- Run `npm install` for `yarn`
- Start server with command `nodemon server`
